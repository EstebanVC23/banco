# Sistema Bancario en Python

Aplicación bancaria sencilla desarrollada en **Python** con interfaz gráfica usando **Tkinter**.

## Archivos del proyecto

- **banco.py**  
  Archivo principal del sistema. Contiene la interfaz gráfica y controla el flujo de la aplicación.  
  **Este es el archivo que se debe ejecutar.**

- **cuentas.py**  
  Maneja la lógica del cliente y las operaciones bancarias (ingresos, retiros y CDT).

- **productos.py**  
  Define los productos bancarios: cuenta de ahorro, cuenta corriente y CDT.

- **Clientes.json**  
  Almacena los clientes válidos del sistema.

- **Cuentas.json**  
  Guarda la información de cuentas, saldos, fechas e intereses.
